## Map 与 WeakMap 区别

| 区别点     | Map                                   | WeakMap                              |
| ---------- | ------------------------------------- | ------------------------------------ |
| 键类型     | 任意类型（对象、基本类型）            | 只能是引用数据类型（不能是基本类型） |
| 键的强引用 | 是，键会被强引用，影响垃圾回收        | 否，键是弱引用，不影响垃圾回收       |
| 可迭代性   | 支持迭代（如 for...of、keys、values） | 不可迭代，不能遍历所有键值           |
| 键值数量   | 可通过 size 属性获取                  | 无 size 属性，无法获取键值数量       |
| 用途       | 通用的键值对存储                      | 适合存储与对象相关的私有数据         |
| 内存管理   | 只要 Map 持有键，键就不会被回收       | 如果没有其他引用，键对象会被自动回收 |
