# 判断一个变量是数组

### 一.Array.isArray() 方法（推荐）

使用 Array.isArray() 可以准确地判断变量是否为数组，这是最常用且最可靠的方法。

```JavaScript
const arr = [1, 2, 3];
console.log(Array.isArray(arr)); // true
```

### 二.instanceof 运算符

使用 instanceof 检查一个变量是否为 Array 的实例。但是，如果数组是在不同的全局上下文（比如 iframe）中创建的，这种方法会失效。

```javascript
console.log(arr instanceof Array); // true
```

### 三.Object.prototype.toString.call

这种方法会返回变量的具体类型字符串（如 "[object Array]"），在判断数组时较为通用，即便跨环境也能准确判断。

```javascript
const arr = [1, 2, 3];
console.log(Object.prototype.toString.call(arr) === "[object Array]"); // true
```

### 四.构造函数判断

可以通过检查 constructor 属性来判断是否是数组，但如果数组的原型被更改过，这种方法会失败。

```javascript
const arr = [1, 2, 3];
console.log(arr.constructor === Array); // true
```
