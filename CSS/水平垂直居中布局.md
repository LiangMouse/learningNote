# CSS 的垂直居中布局

垂直居中是面试以及工程中最常出现的几个布局之一。掌握他非常重要重要 ❗️

一共有子绝父相相对偏移再绝对偏移;子绝父相给定子元素大小后外边距 auto;flex 弹性布局;grid 布局; 等四个方法

## 方法一

先定位后偏移

```CSS
.father {
  position: relative;
}
.son {
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
  // 这里也可以给子元素添加宽高大小，然后设置ml和mt实现相同效果
}
```

## 方法二

根据子元素大小给定宽度

```CSS
.father {
  position: relative;
}
.son {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0px;
  margin: auto;
  height: 100px;
  width: 100px;
  /*  margin的auto需要知道元素的大小来计算得到边距实际值 */
  /* 绝对定位是必要的，四个方向的0触发流体性。除此之外flex */
}
```

## 方法三

给父元素设置`display:flex`，子元素会被应用弹性布局上下文，并在垂直水平方向上居中

```CSS
.father {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

可以使用[place-items](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Reference/Properties/place-items) 属性进行简写

## 方法四

GRID

```CSS
.box {
    width: 200px;
    height: 200px;
    border: 1px solid red;
    display: grid;
}
.children-box {
    width: 100px;
    height: 100px;
    background: yellow;
    margin: auto;
    // grid布局下填满空间，居中定位
}
```
