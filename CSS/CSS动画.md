# CSS动画
一般有两种方式: 过渡动画`transition`和自定义动画`animation`
### transition
| 属性                  | 含义                      |
| ------------------- | ----------------------- |
| transition-duration | 动画的过渡时间，默认是0s,不设置没有过渡效果 |
| transition-delay    | 动画的延迟时间，几秒后触发           |
|                     |                         |
