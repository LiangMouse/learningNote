# CSS 的垂直居中布局

一共有绝对定位,

## 一

先定位后偏移

```CSS
.father {
  position: relative;
}
.son {
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
  // 这里也可以给子元素添加宽高大小，然后设置ml和mt实现相同效果
}
```

## 二

根据子元素大小给定宽度

```CSS
.father {
  position: relative;
}
.son {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0px;
  margin: auto;
  height: 100px;
  width: 100px;
  // margin的auto需要知道元素的大小来计算得到边距实际值
}
```

## 三

无需多言的 FLEX

```CSS
.father {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

## 四

GRID

```CSS
.box {
    width: 200px;
    height: 200px;
    border: 1px solid red;
    display: grid;
}
.children-box {
    width: 100px;
    height: 100px;
    background: yellow;
    margin: auto;
}
```
