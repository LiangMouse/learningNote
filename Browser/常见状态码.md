# 常见 HTTP 状态码

## 概述

1xx: 协议处理的中间状态，还在**请求中**
2xx: 报文已收到并成功处理
3xx: 重定向，资源发生变动，需要重新发送请求
4xx: 客户端错误，请求报文有误，服务器无法处理
5xx: 服务器处理请求时发生错误

### 常见状态码

- 200 - 成功。
- 201 - 创建用户成功
- 204 - 无响应体的请求正确响应
- 301 - 永久重定向（配合 location，浏览器自动处理）。
- 302 - 临时重定向（配合 location，浏览器自动处理）。
- 304 - 协商缓存
- 400 - 客户端请求的语法错误
- 401 - 身份认证有问题
- 403 - 没权限。
- 404 - 资源未找到。
- 409 - 编辑冲突问题
- 500 - 服务端通用错误码。
- 502 - 上层网关报错
- 503 - 服务端超载或者维护状态
- 504 - 网关超时。
