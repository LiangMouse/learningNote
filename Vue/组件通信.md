在 Vue 组件中，参数传递（即数据传递）通常通过 **props**、**事件**、**插槽** 或 **Provide/Inject** 来实现。下面是这些传参方式的简要对比：

| 传参方式               | 适用场景            | 数据流向         | 说明                                       | 示例                            |
| ------------------ | --------------- | ------------ | ---------------------------------------- | ----------------------------- |
| **Props**          | 父组件向子组件传递数据     | 单向数据流（父 → 子） | 父组件通过 `props` 向子组件传递数据，子组件无法修改父组件的数据     | `props: ['message']`          |
| **$emit / Events** | 子组件向父组件传递数据     | 单向数据流（子 → 父） | 子组件通过 `$emit` 触发事件，将数据传递给父组件             | `this.$emit('update', value)` |
| **插槽 (Slots)**     | 在父组件中动态插入子组件的内容 | 内容传递（父 → 子）  | 父组件通过插槽将 HTML 内容传递给子组件渲染                 | `<slot></slot>`               |
| **Provide/Inject** | 跨层级组件传递数据       | 跨层级（不限定父子关系） | 父组件通过 `provide` 提供数据，子组件通过 `inject` 获取数据 | `provide()`, `inject()`       |
| **v-model**        | 双向绑定（父子组件之间）    | 双向数据流（父 ↔ 子） | `v-model` 使父子组件之间双向绑定同一个数据               | `v-model="value"`             |

### 总结：

- **Props**：父组件向子组件单向传递数据。
- **$emit**：子组件向父组件传递数据。
- **插槽**：父组件向子组件传递 HTML 内容。
- **Provide/Inject**：跨层级传递数据，避免层层传递。
- **v-model**：实现双向绑定，父子组件之间的数据同步。
