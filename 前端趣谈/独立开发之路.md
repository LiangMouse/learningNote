# 独立开发之路

## 独立开发理念

需求推动开发， MVP最小功能版本上线

## 推荐技术栈
- Next.js + TypeScript：全栈能力，能快速做出 SSR、SSG 的项目，也适合 SEO；TS 让代码更安全
- TailwindCSS：写样式效率高，配合组件化开发特别舒服；界面好看，效果能直接给雇主看，省下很多扯皮时间。
- Supabase：做数据库和auth认证，省得自己再搭建后端。对于小团队、AI 项目尤其方便，几分钟就能上线数据库。
- LLM / AI 接入：像 OpenAI、Hugging Face API，做 AI 问答、生成、推荐这些场景都用得上。
- Vercel / Netlify：部署超方便，提交完就能给雇主一个链接，直接看效果。
- [nexty.dev](https://nexty.dev/zh): 项目基础设施
