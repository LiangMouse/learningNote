# HTML5 特性

HTML5 并非单一的 “HTML 标记语言” 升级，而是 W3C 定义的一套 “现代 Web 技术标准套件”

- Canvas 标签
- 语义化标签
- 多媒体支持 `<audio>、<video>`
- 浏览器存储 `localSotrage 、 sessionStorage `
- history API：go, forward, back, pushstate
- web worker 多线程解决方案
- web socket 长连接方案
- 离线应用支持（Service Worker、离线缓存，允许无网络访问）

在`HTML5`之后，不再有大版本的改动概念，而是直接进行更新 -> HTML [现行标准](https://htmlspecs.com/)
