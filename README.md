### 为什么写博客

之前用过语雀记笔记，但是不开会员不能被其他人看到并且写起来支持语法没有其他 markdown 编辑器更舒适。
对于不被别人看到这点其实还挺重要的。自说自话的笔记整理和记忆效果远没有对着一个看不见的听众侃侃而谈来得好。这也是国外风靡 **橡皮鸭调试法**的原因吧
另外我不认为一天一篇博客的内容是最佳实现，老实说博客内容大部分取材并写在整理八股题期间，我会逐渐以回答面试官回答读者疑问的形式不断完善博客的内容，根据自己所学拓深拓广，出于八股，不止八股。
虽然我现在做不到分享太有价值的知识，但我会很慷慨的和大家交流，后面也会加上评论功能，也欢迎加`QQ`1979139861 和我玩
